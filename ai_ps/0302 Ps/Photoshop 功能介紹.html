Photoshop 功能介紹

2021-03-02 09:01:56

作品集會是 A4 大小
建議使用: 解析度300ppi、8bit RGB、sRGB、外觀比率 1:1(正方形像素)
 
PS 介面:
  • 可以用 :影像 > 影像旋轉 >>  這裡可以旋轉版面
  • 跟 Ai 一樣可以設定個人工作區
  • :編輯 > 偏好設定 > 介面 >> 將介面改黑色
  • 跟 AI 一樣要善用 圖層控制區
    • 新增圖層會是透明圖層(灰白色格)
    • 快速複製圖層: Ctrl + J
    • 可以用 Alt + 點圖層眼睛 會開啟此眼睛， 並關閉其他所有。再點同樣的眼睛會恢復原樣
    • 如果圖層數量多，可以使用群組功能來整理歸類
    • 可以用不同方式鎖定圖層
    • 如果要製作原始背景，用快捷鍵 Ctrl + Delete
  • 左邊工具區
    • 筆刷 >> 版面上按右鍵可以改筆刷頭
      • 用 [ ] 括號放大縮小筆頭大小
      • CAPSLOCK 可以切換滑鼠形狀
        •  
  • 儲存檔案
    • 用PSD檔案
  • 矩形選取畫面工具
    • 取消選取範圍 Ctrl + D
    • 用 Shift 來加選範圍、Alt 來剪選範圍、Shift + Alt 來只選重疊範圍
  • 使用大多數工具時，如果按 Ctrl，會變移動模式
  • Ctrl + T 將形狀變成類似物件
    • 上方會出現彎曲模式 >> 可調整錨點
  • 魔術棒 容許度高: 選更多範圍、低:選更少範圍
    • 應該是依照顏色跟
  • 存檔
    • 通常如果要再編輯、修改圖， 會用 PSD 檔
    • 如果已經要使用圖盡量用 .png 檔 (會去背)
    • 如果用jpg 檔會含背景顏色
  • 快速選取工具選了之後，可以點 "選取並遮住" 來編輯圖案
    • 可以用智慧型半徑跟調整邊緣筆刷工具來修半透明之物
    • 修圖完可以再新增圖層
  • 編完作品後必須將多餘透明背景去掉
    • :影像 > 修剪 >> 透明像素
